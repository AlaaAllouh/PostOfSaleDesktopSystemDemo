 🖥 Post Of Sale (POS) Desktop System

📌 Overview

This project is a **Point of Sale (POS) desktop application** built using **Windows Forms (.NET)**.
It is designed to help sellers manage daily sales operations by recording transactions and tracking purchased items in an efficient and organized way.

A POS system is widely used in retail businesses to handle transactions, manage inventory, and record sales data in real time.

🚀 Features

* Record and manage sales transactions
* Add purchased items and calculate totals
* Simple and user-friendly interface for sellers
* Store and manage data locally
* Display product and transaction details

🧰 Technologies Used

*C#
* Windows Forms (.NET)
* SQL Server / Local Database
* .NET Framework

 🏗 System Concept

The system simulates a real-world POS environment where:

* The seller inputs purchased items
* The system calculates the total cost
* The transaction is recorded and stored

POS systems are essential for businesses as they combine sales processing with data tracking and inventory management ([Investopedia][1])
